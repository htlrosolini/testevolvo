Sobre

Bem vindo a minha aplicação de teste, a mesma foi desenvolvida utilizando as ferramentas:

  • ASP.NET Core e C# para códigos do lado do servidor
  • Angular e TypeScript para códigos do lado do cliente
  • Bootstrap para layout e estilização

Para te ajudar a iniciar eu também criei as seguintes informações

  • Relação de caminhões gravados no banco. Clicando no link Lista de Caminhões você consegue vizualizar a tabela de caminhões no banco
  • Criar Caminhão. Para criar um caminhão basta preencher e salvar o formulario no link Criar Caminhão
  • Editar Caminhão. Para editar algum caminhão na Lista de Caminhões existem os botões editar e deletar em cada linha é só usar o botão de editar e alterar os dados que necessita no formulario
  • Excluir Caminhão. Para excluir algum caminhão na Lista de Caminhões existem os botões editar e deletar em cada linha é só usar o botão de excluir e confirmar a exclusão
